# Municipal Complaint Management Portal

A Django-based web application for managing and tracking public complaints efficiently.

## Features

- User Registration and Login
- Complaint Submission
- Officer Management
- Complaint Status Tracking
- Admin Dashboard
- File/Image Upload Support

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite3

## Project Structure

- complaints
- officers                                                                                                                                                      
- users
- templates
- media

## Run the Project

```bash
python manage.py runserver
```
